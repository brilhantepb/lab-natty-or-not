```markdown
# Podcast: Mergulhando no Mundo das Redes - Protocolos e Endereçamento IP ;)

## 📒 Descrição
Este é um podcast de 13 minutos criado para explicar dois capítulos de um pdf do Senac-SP sobre, entre outros assuntos nele abordados, Protocolos e Endereçãmento IP. Elaborado pelo prefessor Walter Augusto Varella, é intitulado "Infraestrutura de TI".

## 🤖 Tecnologias Utilizadas
IA Genertiva NotebookLM, do Google

## 🧐 Processo de Criação
Usei exlusivamente a IA NotebokLM para estruturar o Podcast, sem qualquer trilha musical ao fundo. A ideia é sentir-se em uma sala de aula.

## 🚀 Resultados
O objetivo principal foi alcançado, pois todo o dinamismo usado pelos narradores contribuiu sobremaneira para a compreensão dos dois capítulos do PDF

## 💭 Reflexão (Opcional)
Este foi o meu primeiro trabalho elaborado com o auxílio de uma IA. O NotebookLM ainda não está plenamente configurado para produzir áudios em português. Mesmo que eu tenha instruído isso no prompt, a voz ainda trouxe uma mescla com o inglês e alguns trechos ficaram com a audição prejudicada. Ainda assim, a experiência foi positiva e considero a ferramenta excelente na aplicação ao aprendizado.
```
